Welcome to the SynapseIntegrateSocialMedia repository.

In this project you can find the files used to show how to use Azure Synapse Integrate pipelines to get data from social media platforms and store the data for analytics.
The original blog is posted in the Microsoft Tech Community: <a href="https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-pipelines-for-social-media-youtube-example/ba-p/2615156" target="_blank">Azure Synapse (Pipelines) for Social Media – YouTube example</a>.
